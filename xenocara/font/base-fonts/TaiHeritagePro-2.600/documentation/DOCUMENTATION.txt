DOCUMENTATION
Tai Heritage Pro
========================

This file describes the documentation files included with the 
Tai Heritage Pro font family. This information should be distributed 
along with the Tai Heritage Pro fonts and any derivative works.

Primary documentation for the Tai Heritage Pro font family is on the 
Tai Heritage Pro website (http://software.sil.org/taiheritage/).

The TaiHeritagePro-features.pdf file describes the use of Graphite 
features that are in the font.

The TaiHeritagePro-features.odt file illustrates the use of the 
Graphite features, and is source file for the pdf.
