Dai Banna SIL - Versions and Changes

Announcement list

If you would like stay informed of updates to Dai Banna SIL and other SIL fonts, please subscribe to the SIL Font News Announcement List. For more information see About.

Our font announcements are also available through Twitter [@silfonts](http://twitter.com/silfonts).

Current version

The latest version of the fonts is always available from the Dai Banna SIL download page as a .zip archive for all major platforms.

This font package is also available in the SIL software repository for Ubuntu. Users can subscribe to this software repository and get current versions and updates automatically.

Previous versions

Previous versions remain available from the Dai Banna SIL download page.

Change history

A detailed list of changes for each version of the fonts is on the Version history page.
